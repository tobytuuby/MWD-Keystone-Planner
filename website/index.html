How It Works

The flow is similar to the original MR Helper site, updated for your new bot and slash command.

1. Enter Character Details

Use the command with a character path and optional flags.

/mwd-kp eu/argent-dawn/ellorett --simulate 15

2. Bot Pulls Raider.IO Data

The bot reads best runs, fills missing dungeons, and calculates score opportunity for each dungeon.

3. Sorted Recommendations

Results are sorted by potential score increase so players can pick the most valuable runs first.

4. Visual Output

The bot returns a generated image summarizing current score, target levels, and projected score gains.
